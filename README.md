## 芦笋录屏是什么？

芦笋录屏是新一代的录屏工具，跟传统录屏工具相比，芦笋不仅可以录制屏幕，它还可以同时录制「人像+屏幕」，此外芦笋录屏还提供了美颜、虚拟背景、绿幕抠像、轻剪辑等更多高级功能。你可以将录制好的视频，通过链接分享给 Ta，Ta 可以一键访问链接通过电脑或手机随时查看和发表评论。如果你想体验芦笋录屏，可通过以下官网链接访问
